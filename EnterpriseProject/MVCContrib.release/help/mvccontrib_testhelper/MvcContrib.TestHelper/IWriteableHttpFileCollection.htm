Type IWriteableHttpFileCollection

Namespace MvcContrib.TestHelper

Defines an extended HttpFileCollectionBase which allows modifications to assist with testing.
Using this interface allows us to work around that fact that the indexers on HttpPostedFileBaseare read only and cannot be changed by overrides.
