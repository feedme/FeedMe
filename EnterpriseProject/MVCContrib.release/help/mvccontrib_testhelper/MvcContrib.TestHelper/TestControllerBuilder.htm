Type TestControllerBuilder

Namespace MvcContrib.TestHelper

This is primary class used to create controllers. After initializing, call InitializeController to create a controller with proper environment elements. Exposed properties such as Form, QueryString, and HttpContext allow access to text environment. RenderViewData and RedirectToActionData record those methods
